Structural Engineering -- Load-Bearing Elements

The Eight Core Principles

These walls carry the weight of everything above. Remove any one and the structure fails.

1. Context Is the Primary Constraint

Claude's context window (~200K tokens) is finite. Quality degrades as it fills. This is the load limit -- every pattern optimizes around it.

2. External State > Internal Memory

Store state in files, git, and databases. External state survives when sessions collapse. The building's records live outside any one room.

3. Fresh Context > Extended Sessions

Quality of 20 five-minute sessions exceeds one 100-minute session. Each floor starts with clean materials, not leftover debris.

4. The Iron Law: Orchestrators coordinate. Workers execute. Never mix.

5. Atomic, Verifiable Tasks

Break into smallest testable units. Each task fits in one load-bearing member. Explicit acceptance criteria for every connection point.

6. Quality Gates Before Commit

Never pour the next floor until the current one has cured. Typecheck, test, lint -- then commit.

7. Isolation for Parallel Work

Git worktrees provide separate construction zones. Different crews, different sites, no collisions.

8. Explicit Communication Channels

Progress files, structured definitions, clear acceptance criteria. The blueprints must be readable by every crew member.

Site Safety -- Structural Failures

What Brings Buildings Down

Hazard: Context Rot

Single long sessions where the foundation crumbles. Quality degrades, decisions forgotten, the structure becomes unsafe. Fix: Fresh context per iteration. Tear down the scaffolding, pour a new footing.

Hazard: Orchestrators Writing Code

The site foreman picks up a welding torch. Loses oversight. Context fills with implementation debris. Fix: The Iron Law. The foreman supervises. The welder welds.

Hazard: Missing Verification

Marking a floor "complete" without inspection. Cracks compound. One bad pour becomes a condemned building. Fix: Quality gates. Always inspect before the next floor begins.

#!/bin/bash
MAX_ITERATIONS=${1:-25}

for i in $(seq 1 $MAX_ITERATIONS); do
  echo "=== Iteration $i ==="
