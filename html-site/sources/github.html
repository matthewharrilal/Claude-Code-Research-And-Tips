CC Mirror - Multi-Agent Orchestration

Quick Win github.com/numman-ali/cc-mirror @nummanali

CC Mirror unlocks Claude Code's disabled multi-agent orchestration feature by creating an isolated variant environment. The feature exists within Claude Code's codebase but is disabled by default.

Native Task Management API Enabled:

  • TaskCreate - Create with subject, description, acceptance criteria
  • TaskList - Filter by status, owner, blocked state
  • TaskGet - Full context with description, comments, dependencies
  • TaskUpdate - Claim, add comments, resolve, link references
  • addBlocks / addBlockedBy - Dependency management

Installation

# Step 1: Create isolated variant
npx cc-mirror quick --provider mirror --name mclaude

# Step 2: Launch it
mclaude

# Step 3: In the terminal
"Load the orchestration skill and initiate"

Provider Options

# Mirror provider (default - uses Claude's internal primitives)
npx cc-mirror quick --provider mirror --name mclaude

# GLM-4.7 provider (good multi-agent coordination)
npx cc-mirror quick --provider glm --name glm-claude

# OpenRouter provider
npx cc-mirror quick --provider openrouter --name or-claude

# LiteLLM provider
npx cc-mirror quick --provider litellm --name lite-claude

Checkpoint: CC Mirror Working

After installation, you should see:

  • Agent task list with checkbox status indicators
  • Dependency tracking with [!] for blocked tasks
  • Background agent monitoring view

Gas Town - Agent Factory

Deep Dive github.com/steveyegge/gastown @steve_yegge

Gas Town is a factory for coding agents, not a single improved agent. While everyone focuses on making their "ant" run longer, Steve Yegge argues nature prefers colonies.

The 8 Stages of Dev Evolution

StageDescription
1Zero/near-zero AI
2Agent in IDE, permissions on
3Agent in IDE, YOLO mode
4Agent fills screen, code is just diffs
5CLI single agent, YOLO
6CLI multi-agent (3-5 parallel)
710+ agents, hand-managed
8Building your own orchestrator

Warning: Stage 7+ Only

Gas Town is for Stage 7+ developers. If you're not there, "you'll get your face ripped off by superintelligent chimpanzees."

7 Worker Roles

RoleFunction
MayorTown-level coordination
DeaconMonitoring/handshakes
DogsQuality gates/watchdogs
RefineryTask decomposition
PolecatNamed persistent workers
WitnessPer-rig observer
CrewEphemeral workers

Installation

# Install
go install github.com/steveyegge/gastown/cmd/gt@latest

# Initialize town
gt init

# Add rig (project repo)
gt rig add <repo-path>

# Launch roles
gt mayor up
gt refinery up beads
gt polecat up beads alice

Community Extensions

  • GTGUI - Age of Empires style GUI (github.com/TRIBE-INC/gtgui)

Community Plugins

Claude-Mem - Persistent Memory

Medium Lift github.com/thedotmack/claude-mem

Gives Claude Code "infinite memory" by automatically capturing sessions, compressing to semantic summaries, and injecting relevant context into future sessions.

MetricImprovement
Token usageUp to 95% fewer per session
Tool calls before limit~20x more
StorageLocal SQLite only
/plugin marketplace add thedotmack/claude-mem
/plugin install claude-mem

Claude HUD - Real-Time Visibility

Quick Win github.com/jarrodwatts/claude-hud 1.8K stars

Shows context remaining (color-coded), tool execution status, subagent activity, todo progress, and session duration.

/plugin marketplace add jarrodwatts/claude-hud
/plugin install claude-hud
/claude-hud:setup

Sample output:

[Opus 4.5] 35% | 1 MCPs | 1m
Read x2 | WebFetch x1 | Skill x1 | Edit x1

Ralph Wiggum Plugin

Medium Lift claude-plugins-official

Goal-conditioned loop that keeps Claude Code running until a verifiable goal is reached. Uses stop hooks to intercept session exits and re-inject prompts.

/plugin
# Type: ralph

# Or full command:
/plugin install ralph-wiggum@claude-plugins-official

Wrapper Scripts & Utilities

rpai - tmux Multi-Agent Manager

Quick Win @radoslav11

TUI tool for tmux that makes monitoring and switching between multiple parallel AI agents actually usable.

cargo install rpai
rpai  # In any tmux session

System - Remote Mac Control

Medium Lift github.com/ygwyg/system

Open-source agent turning your Mac into an API. Chat from your phone, Claude executes commands via secure Cloudflare tunnel.

git clone https://github.com/ygwyg/system
cd system
npm install
npm run setup  # Interactive wizard
npm start      # Launch bridge + tunnel
