Techniques and Capabilities

You Are Here

This section covers the building blocks of Claude Code: Skills (Markdown instruction sets), MCP Servers (external tool integration), Hooks (event-driven automation), Security (permissions and sandboxing), and Mobile/Async workflows. Master these techniques to build sophisticated agent systems.

Skills and Custom Commands
Extend Claude Code with Markdown-based instruction sets. Create personas, workflows, and reusable recipes that load on demand.

MCP Servers
Connect Claude to external systems via Model Context Protocol. Databases, APIs, files, Git, Slack - all through a standard interface.
MCP Integration Tools
Hooks
Event-driven automation with shell scripts. Auto-format code, send notifications, block unsafe operations, create guardrails.
Hooks Automation Events
Security
Understand the trust model, permission levels, sandboxing options. Keep your system safe while leveraging Claude's power.
Security Permissions Sandboxing
