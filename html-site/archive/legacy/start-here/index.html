Key Mental Models

The foundational frameworks that advanced practitioners use to think about Claude Code.

📚
Context is Constraint
What you put in CLAUDE.md shapes what's possible. Good context enables; poor context confuses. Think of it as programming the agent's worldview.
♺
External State > Internal Memory
Don't rely on Claude "remembering." Write state to files. progress.txt, prd.json, AGENTS.md - make it explicit and persistent.
🌱
Fresh > Extended
Long sessions accumulate confusion. Fresh context windows are clean slates. The Ralph loop pattern exploits this insight.
🔍
Verify, Don't Trust
Always verify Claude's work. Run tests. Check file changes. The verification loop is what separates safe usage from disaster.
