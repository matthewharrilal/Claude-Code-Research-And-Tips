Architecture Journey

You Are Here

What this journey covers: System design and pattern selection for Claude Code deployments. You'll learn the full complexity ladder - from single-session vanilla Claude to enterprise-scale swarms with hundreds of agents - and develop the judgment to choose the right level for any project.

Who this is for: Developers ready to move beyond basic Claude Code usage. You should already understand the mental models (context as constraint, external state over memory) before diving into architecture.

What you'll be able to do:

  • Select the right architecture pattern for any project scale
  • Understand unlock conditions for each complexity level
  • Design multi-agent systems with appropriate coordination
  • Manage context composition for large codebases

Quick Reference

PATTERN SELECTION QUICK GUIDE:

1 agent, at keyboard         → Single Session (Vanilla Claude)
1 agent, walk away           → Ralph Loop
2-5 agents, independent work → Parallel Ralph
3-10 agents, human judgment  → Claude Squad
5-10 agents, automated       → CC Mirror
10-30 agents, factory scale  → Gas Town
30-100 agents, self-learning → SONA Swarm
100+ agents, compliance      → Enterprise Stack
