Learning Journeys

You Are Here

What this is: The Journey Hub is your launchpad to structured learning paths through the Claude Code knowledge base. Instead of reading reference documentation linearly, these journeys guide you through concepts in an order optimized for understanding and skill-building.

How to use this: Choose the journey that matches your current learning goal. Each journey builds from foundational concepts to advanced applications. You can also jump between journeys as needed - they're designed to complement each other.
