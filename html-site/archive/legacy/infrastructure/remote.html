System - Remote Mac Control via Claude AI

Your Mac becomes an API - chat from your phone, Claude executes commands

You Are Here

Context: This covers "System" - an open-source agent that turns your Mac into an API. You chat from anywhere, Claude executes commands on your machine via secure Cloudflare tunnel.

Where this fits: Perfect for always-on Mac Mini setups, personal automation infrastructure, and remote control while mobile.

Architecture

+-------------------------------------+
|  BRAIN (Cloudflare Worker)          |
|  +-- Claude AI for NLP              |
|  +-- State in Durable Objects       |
|  +-- Scheduling (cron/natural lang) |
|  +-- Tool orchestration             |
+-------------------------------------+
              |
    Cloudflare Tunnel (secure, no ports exposed)
              |
              v
+-------------------------------------+
|  BODY (Local Mac Bridge)            |
|  +-- Express server                 |
|  +-- AppleScript execution          |
|  +-- Safe shell commands (allowlist)|
|  +-- Raycast extension deep links   |
+-------------------------------------+
Key security: No ports exposed, no open endpoints - just a secure pipe only the agent can use.

What It Can Do

Media

  • Play music (Apple Music/Spotify)
  • Control playback

Communication

  • Send texts (iMessages)
  • Email actions

Productivity

  • Create Linear issues
  • Run Shortcuts

System

  • Execute AppleScript
  • Trigger Raycast extensions
  • Schedule recurring tasks

All via natural language from anywhere.

Setup

git clone https://github.com/ygwyg/system
cd system
npm install
npm run setup  # Interactive wizard
npm start      # Launch bridge + tunnel

Setup Wizard Handles:

  • Scans Raycast extensions
  • Configures Anthropic API key
  • Chooses UI/API mode
  • Sets local/remote access

Checkpoint: System Running

You should see:

  • Bridge server running locally
  • Cloudflare tunnel connected
  • API endpoint available

Prerequisites

  • macOS
  • Node.js 18+
  • Anthropic API key
  • Raycast (optional, for extensions)
  • cloudflared (brew install cloudflared)

Caveats & Gotchas

Mac must be online

Remote access requires your Mac to be powered on and connected to the internet. Consider an always-on Mac Mini.

Raycast extensions with UI/forms

Extensions requiring interactive UI elements may not work perfectly through the API.

OAuth-based extensions

Need one manual Raycast run first to establish authentication.

Quick mode tunnel is ephemeral

For permanent deployment, needs Workers deploy instead of quick tunnel.
