Tooling

Plugins and tools that extend Claude Code's capabilities. Each tool solves a specific problem in the Claude Code workflow.

rpai - tmux Multi-Agent Manager

TUI dashboard for monitoring and switching between multiple AI agents running in tmux panes.

Quick Win tmux multi-agent rust

Claude-Mem - Persistent Memory

"Infinite memory" plugin that captures sessions, compresses to semantic summaries, and injects context into future sessions.

Medium Lift memory context plugin

Claude HUD - Session Visibility

Real-time dashboard showing context usage, tool activity, subagent status, and todo progress.

Quick Win visibility context plugin

Claude Agent SDK

Programmatic SDK for building autonomous agents with computer control. Foundation for embedded products.

Medium Lift SDK agents python
