The Creator

Creator of Claude Code
Boris Cherny
Staff Engineer at Anthropic
The canonical reference for Claude Code usage. His "surprisingly vanilla" setup proves the tool works out of the box. Learn the 13-point workflow, Plan Mode, verification loops, and the compounding CLAUDE.md system.

Pattern Pioneers

Ralph Pattern
Matt Pocock & Ryan Carson
TypeScript Educator & Treehouse Founder
Pioneers of the Ralph Wiggum Loop. The 3-file system (PROMPT.md, LOG.md, CONTEXT.md), pre-commit hook discipline, and "let it cook" philosophy. Essential for autonomous overnight development.
Enterprise Scale
Shrivu Shankar
Enterprise Architect, Billion-Token Scale
13KB CLAUDE.md architecture, Master-Clone pattern, and "shoot and forget" philosophy. Part of a team consuming several billion tokens per month for codegen. Enterprise-grade workflow patterns.
Context Architect
Ashley Ha
Context Management Practitioner
60% context rule and 4-phase workflow methodology. The thoughts/ directory pattern for persistent Claude memory. Systematic approach to context window optimization.
Orchestration Architect
Addy Osmani
Engineering Leader, Chrome Team
Conductor vs Orchestrator paradigm for AI agent architectures. Overnight coding workflow for time-shifted productivity. Elevate Newsletter on AI-assisted development.
