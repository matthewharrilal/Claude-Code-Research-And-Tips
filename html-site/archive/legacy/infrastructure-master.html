Sandboxing

Isolated Execution Environments

Running Claude Code in sandboxed environments protects your system from unintended side effects.

Options

MethodIsolationSetup
DockerContainerMedium
VMFullHigh
CloudflareWorkerLow
NixDeclarativeMedium

Docker Example

docker run -it --rm \
  -v $(pwd):/workspace \
  -w /workspace \
  claude-code-sandbox

Deep dive: See Sandboxing Guide

Remote Access

Access Claude Code From Anywhere

Run Claude Code on a remote machine and access it from any device.

SSH + tmux Pattern

# On remote machine
tmux new -s claude

# From any device
ssh user@remote -t "tmux attach -t claude"

Cloudflare Tunnel

Expose your development environment securely without opening ports:

cloudflared tunnel --url localhost:8080

Deep dive: See Remote Access Guide

Chrome MCP

Browser Automation

Understanding how Claude Code interacts with browsers through MCP and Chrome DevTools Protocol.

How It Works

  1. MCP server receives browser commands
  2. Translates to Chrome DevTools Protocol (CDP)
  3. Executes in headless or visible Chrome
  4. Returns results to Claude

Use Cases

  • Web scraping
  • Automated testing
  • Form filling
  • Screenshot capture

Deep dive: See Chrome MCP Guide
