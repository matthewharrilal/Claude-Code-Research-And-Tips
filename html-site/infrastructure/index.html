You Are Here: The Infrastructure section covers deployment, remote access, and cloud configurations for Claude Code. When you need to run Claude Code somewhere other than your local terminal.

When to Use: Setting up cloud VMs, configuring remote access via SSH, deploying for team usage, or running Claude Code from mobile devices.

Infrastructure

Deployment and remote access patterns. Run Claude Code anywhere - cloud VMs, headless servers, mobile-triggered workflows.

Why Infrastructure Matters

Claude Code's power increases dramatically when you can run it:

  • Remotely: VMs that never sleep, always-on development environments
  • In Parallel: Multiple instances on different machines
  • From Anywhere: Mobile access via SSH clients
  • At Scale: Team deployments with shared configurations
