# Font Identification Results: Sanrok Studio Typography

## Executive Summary

**CONFIRMED: Sanrok Studio uses Instrument Serif for its main typography**, paired with Inter as the body font. This was verified by directly analyzing the Sanrok website (sanrokstudio.com).

However, the **display typography in the reference images** shows characteristics of a **Didone/Modern serif** that differs from Instrument Serif - possibly a custom or modified version, or additional typefaces used in their design work. The reference display font features:

- Very high stroke contrast (hairline horizontals, thick verticals)
- Unbracketed serifs (no curves at serif junctions)
- Ball terminals on curved letters
- Vertical stress
- Both filled and outline versions
- Roman and italic weights

### About Instrument Serif

**Foundry:** Instrument (design agency)
**Designers:** Rodrigo Fuenzalida, with direction from Jordan Egstad, JD Hooge, Jack De Caluwe
**Released:** 2023
**Availability:** FREE (Google Fonts, SIL Open Font License)
**Styles:** Roman + Italic only

> "Instrument Serif is a condensed display font designed for the Instrument brand. It is intended for use at large sizes and offers a contemporary take on some of the classic serif characteristics."

**Key Characteristics:**
- Condensed display font
- Contemporary interpretation of old style serifs
- Designed specifically for Instrument brand
- 374 Latin glyphs supporting 86 languages
- Available on [Google Fonts](https://fonts.google.com/specimen/Instrument+Serif) and [Adobe Fonts](https://fonts.adobe.com/fonts/instrument-serif)

**Why Sanrok Chose It:**
- Free and open source
- Modern elegance without being too ornate
- Works well at display sizes
- Pairs beautifully with Inter

## Part 4: Creating Outline/Stroke Versions

The Sanrok reference shows both filled and outline typography. Most fonts do not include outline versions, but you can create them with CSS.

### CSS Text Stroke Method

```css
/* Basic outline text */
.outline-text {
  -webkit-text-stroke: 1px #000000;
  -webkit-text-fill-color: transparent;
  color: #000000; /* Fallback */
}

/* Thicker outline */
.outline-text-thick {
  -webkit-text-stroke: 2px #000000;
  -webkit-text-fill-color: transparent;
}

/* With paint-order for cleaner strokes */
.outline-text-clean {
  -webkit-text-stroke: 1.5px #000000;
  -webkit-text-fill-color: transparent;
  paint-order: stroke fill;
}

/* Gradient outline */
.outline-gradient {
  -webkit-text-stroke: 2px transparent;
  -webkit-text-fill-color: transparent;
  background: linear-gradient(135deg, #ff0000, #0000ff);
  -webkit-background-clip: text;
  background-clip: text;
}
```

### Browser Support Note

> "The `-webkit-text-stroke` property is only supported by WebKit-based browsers. The stroke is aligned to the center of the text shape (as in Adobe Illustrator), with no option for inside or outside alignment."

### Alternative: Text Shadow Method

```css
/* Outline via multiple shadows (better browser support) */
.outline-shadow {
  color: transparent;
  text-shadow:
    -1px -1px 0 #000,
     1px -1px 0 #000,
    -1px  1px 0 #000,
     1px  1px 0 #000,
     0   -1px 0 #000,
     0    1px 0 #000,
    -1px  0   0 #000,
     1px  0   0 #000;
}
```

## Part 5: Implementation Recommendations

### For Replicating Sanrok Website Exactly

```html
<!-- Google Fonts -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Instrument+Serif:ital@0;1&family=Inter:ital,wght@0,400;0,500;0,700;0,800;0,900;1,400;1,500;1,700&display=swap" rel="stylesheet">
```

```css
:root {
  --font-display: 'Instrument Serif', Georgia, serif;
  --font-body: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
}

h1, h2, .headline {
  font-family: var(--font-display);
}

body, p {
  font-family: var(--font-body);
}
```

### For Replicating Didone Display Style

```html
<!-- Bodoni Moda (closest free match) -->
<link href="https://fonts.googleapis.com/css2?family=Bodoni+Moda:ital,opsz,wght@0,6..96,400;0,6..96,700;0,6..96,900;1,6..96,400;1,6..96,700&display=swap" rel="stylesheet">
```

```css
:root {
  --font-didone: 'Bodoni Moda', Georgia, serif;
}

.hero-headline {
  font-family: var(--font-didone);
  font-weight: 900;
  font-optical-sizing: auto;
}

.hero-headline-outline {
  font-family: var(--font-didone);
  font-weight: 700;
  -webkit-text-stroke: 1.5px currentColor;
  -webkit-text-fill-color: transparent;
}
```
