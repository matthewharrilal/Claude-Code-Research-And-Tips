- **Stroke Contrast:** Extreme (classic Didone)
- **Serifs:** Hairline, slightly rounded
- **R Leg:** Curved calligraphic tail
- **G Spur:** Present
- **Proportions:** Wide, bold display face
- **Ball Terminals:** Large, distinctive

### Distinctive Features
- **"Fatface" weight** - extremely bold, poster-style
- **Rounded terminals** - softer than sharp Didones
- **Wide proportions** - generous letter spacing built-in
- **19th century advertising poster influence** - theatrical, attention-grabbing
- **Refined small caps** - unusual for a display font

## Initial Assessment

### Match Quality vs Sanrok Reference: 2.5/5

**Key Similarities:**
- Extreme stroke contrast
- Didone genre characteristics
- Display-oriented design
- High visual impact

**Key Differences:**
- **Much bolder** than the reference - Fatface is an extra-black weight
- **Warmer, softer** character vs. sharp editorial feel
- **Wider proportions** - reference is more condensed
- **More theatrical** - reference is more refined/sophisticated
- **Single weight only** - no flexibility
- **No italic** - limiting for editorial use

### Best Use Case
Large, impactful headlines where maximum visual weight is desired:
- Magazine covers
- Poster titles
- Hero sections requiring gravitas
- Luxury/fashion when theatrical impact is needed

### Limitations
- **Too bold** for the Sanrok aesthetic
- **Single weight** - no typographic hierarchy possible
- **No italic** - can't create emphasis within serif text
- **Display only** - definitely not for body text

### Comparison Notes
Abril Fatface is a beautiful font, but it's solving a different problem than the Sanrok reference. The Sanrok aesthetic calls for sharp, refined, contemporary editorial Didone. Abril Fatface is bold, warm, and theatrical - more suited to vintage-inspired luxury or dramatic poster work.

**Recommendation:** Not recommended for Sanrok-style work. Consider only if the design direction shifts toward bolder, more theatrical typography.
