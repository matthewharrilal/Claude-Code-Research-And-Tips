# Font Comparison Ranking Summary

## Executive Summary

This document summarizes the letter-by-letter comparison of five font candidates against the Sanrok Studio reference typography. The analysis evaluated each font on 8 key letterforms (R, S, G, A, W, E, O, K) using a 1-5 scoring system.

**Reference Characteristics (Target):**
- Stroke contrast: 1:8 to 1:10 (extreme)
- Serifs: Unbracketed hairline
- Stress: Vertical
- R leg: Curved/calligraphic (NOT straight)
- G spur: Has horizontal spur extending into counter
- Terminals: Horizontal shear on S, C
- Apex (A, W): Sharp, pointed
- Classification: Didone/Modern display serif

## Recommendations by Use Case

### For Maximum Accuracy to Reference Design

**Use: Right Didone (Tight or Narrow subfamily)**

- Closest match to contemporary editorial aesthetic
- Multiple width options for precise proportions
- Includes outline styles
- G spur present
- Modern interpretation aligns with reference philosophy

**Trade-off:** Commercial licensing required (~$30-150+ depending on scope)

### For Matching Sanrok's Actual Website

**Use: Instrument Serif + Inter**

- This is what Sanrok actually uses on sanrokstudio.com
- Free on Google Fonts
- Condensed, contemporary feel
- Works well for display sizes

**Trade-off:** Single weight only, different classification (old style vs. Didone)

### For Best FREE Match to Reference

**Use: Bodoni Moda**

- Closest free Didone
- Extreme contrast matches
- Multiple weights + optical sizes
- Variable font support

**Trade-off:** Missing G spur, more traditional/scholarly than contemporary

### For Bold Hero Moments (Free)

**Use: Abril Fatface**

- Maximum impact
- Maintains hairline serifs despite bold weight
- Sharp apexes

**Trade-off:** Single (fat face) weight only, no italic, poster aesthetic vs. editorial

## Conclusion

**Winner: Right Didone** achieves the highest match score (34/40) by combining contemporary Didone characteristics with the editorial edge of the reference. Its multiple width subfamilies provide flexibility that other candidates lack.

**Best Free Alternative: Bodoni Moda** offers the most complete free Didone experience, though it lacks the G spur and reads more traditionally than the reference.

**Special Case: Instrument Serif** is the actual font from Sanrok's website, making it the most authentic choice despite being classified as old style rather than Didone.
