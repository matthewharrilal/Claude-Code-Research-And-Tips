Card Component

A comprehensive guide to implementing the card component with halftone image treatment and editorial typography.

Overview

The card component displays project previews with a distinctive halftone image treatment. It uses the Image-Top layout structure established in Experiment 2, placing the image as the dominant visual element.

Note

This component requires the halftone-large.svg pattern file to be available in your assets directory.

Required Files

src/
├─components/
│└─Card.tsx
└─assets/
└─patterns/
└─halftone-large.svg

Implementation

The card uses CSS pseudo-elements for the halftone treatment. This approach avoids server-side image processing while achieving visible texture.

Card.tsx 
export function Card({ title, year, image, tags }) {
  return (
    <article className="card">
      <div className="card__media">
        <img src={image} alt="" />
      </div>
      <div className="card__content">
        <h3>{title}</h3>
        <span>{year}</span>
      </div>
      <div className="card__tags">
        {tags.map(tag => <span key={tag}>{tag}</span>)}
      </div>
    </article>
  );
}

Tip

Use aspect-ratio: 4/3 on the image to maintain consistent proportions across different image sources.

Warning

The halftone pattern size must match the SVG viewBox. Using 16px for a 16x16 pattern prevents visible seams.

Getting Started with Card Components

Follow these steps to implement the card component in your project.

Step 1: Set up the project structure

Create the necessary directories and files for the card component.

Create components directory

mkdir -p src/components

Create Card.tsx file

touch src/components/Card.tsx

Add halftone pattern to assets

Copy halftone-large.svg to src/assets/patterns/

Step 2: Implement the component

Create the React component with the halftone image treatment.

Tip

Start with the HTML structure first, then add the CSS for the halftone effect. This makes debugging easier.

Card.tsx 
import './card.css';

export function Card({ title, year, image }) {
  return (
    <article className="card">
      <div className="card__media">
        <img src={image} alt="" />
      </div>
      <h3>{title}</h3>
      <span>{year}</span>
    </article>
  );
}

Warning

Don't forget to set object-fit: cover on the image to prevent distortion with different aspect ratios.

Step 3: Add the CSS styles

Apply the halftone treatment using CSS pseudo-elements.

card.css 
.card__media {
  position: relative;
}

.card__media::after {
  content: '';
  position: absolute;
  inset: 0;
  background-image: url('halftone-large.svg');
  background-size: 16px 16px;
  mix-blend-mode: multiply;
}
