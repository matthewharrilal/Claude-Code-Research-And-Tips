Pair 3: Gotcha + Code

Adjacency: Vertical (warning before code)

Warning

The halftone SVG pattern must be sized correctly. Using background-size values that don't align with the pattern tile will create visible seams.

halftone.css 
/* Correct: 16px matches SVG viewBox */
.card-image::after {
  background-size: 16px 16px;
}

/* Incorrect: causes visible seams */
.card-image::after {
  background-size: 15px 15px;
}

Pair 5: Essence + Prose

Adjacency: Vertical (takeaway in text flow)

The visible process principle suggests that design systems should reveal their constraints rather than hide them. When users can see how something was made, they understand its boundaries.

The Key Insight

"Constraints create character. Character creates recognition. Recognition creates brand."

This is why halftone dots work so well — they're an honest representation of the printing process, not a decoration applied after the fact.
