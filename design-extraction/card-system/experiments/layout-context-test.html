Layout Context Protocol

Testing integrated card (Exp 1-5 winners) in multiple layout contexts

CRITICAL: Per Plan v2 Audit Every prototype must be evaluated in layout contexts, not just isolation. This page tests the integrated card (A-LARGE halftone + Image-Top + Red Italic Sentence Case) in:
  1. Horizontal Scroll — 5 cards in row, peripheral vision test
  2. Dense Grid — 9 cards in 3x3, cooperation vs competition
  3. Mixed Context — Card next to prose/code, presence test
  4. Axis Transition — H-scroll flowing into grid
