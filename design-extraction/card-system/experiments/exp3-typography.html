Experiment 3 — Typography Reduction

Typography Emphasis Options

Testing different emphasis combinations to achieve "Typography Restraint" while maintaining brand recognition. Current system uses 3 emphases (Red + Italic + Uppercase). Each variant removes one or more.

GRID TEST PROTOCOL: View each 6-card grid for 30 seconds. Ask: Does the grid feel CALM or COMPETITIVE? Does typography SERVE the image or FIGHT it? After viewing all variants, which would you recognize as "that system"?
