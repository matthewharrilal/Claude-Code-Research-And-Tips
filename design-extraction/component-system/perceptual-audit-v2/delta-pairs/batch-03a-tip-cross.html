C.6: Tip Callout + Core Abstraction
Tip
Wrap third-party libraries in your own abstractions. This makes them easier to swap out later if needed.
Core Principle
"Every external dependency is a liability. Wrap it, contain it, control it."
// Your abstraction
export const httpClient = {
  get: (url) => axios.get(url),
  post: (url, data) => axios.post(url, data)
};

C.8: Tip Callout + Reasoning Component
Tip
Consider both options carefully before deciding. There's rarely a universally "correct" answer.
Should we use REST or GraphQL for our API?
REST
Simple, cacheable, widely understood. Better for CRUD operations.
GraphQL
Flexible queries, single endpoint. Better for complex data requirements.
Recommendation
Start with REST for simplicity, migrate to GraphQL only if data needs become complex.
