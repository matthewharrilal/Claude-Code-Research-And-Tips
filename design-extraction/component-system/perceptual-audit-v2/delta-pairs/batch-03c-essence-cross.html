E.6: Essence Callout + Reasoning Component
Essence
"Reasoning without constraints is wandering. Constraints are what make decisions possible."
Should we build or buy this capability?
Build
Full control, but requires expertise and maintenance. Core competency matters here.
Buy
Faster to market, but creates dependency. Acceptable for non-core capabilities.
Constraint-Based Decision
Ask: "Is this our competitive advantage?" If yes, build. If no, buy and move on.
