File Tree — In Context

When setting up a new Claude Code project, the file structure matters. Claude reads CLAUDE.md at the project root to understand context and constraints. 

your-project/
├── CLAUDE.md
├── .claude/
│   └── settings.json
└── src/
    └── ...

The .claude/ directory contains project-specific settings that persist across sessions.
