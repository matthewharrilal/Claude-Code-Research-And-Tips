## 7 PERCEPTUAL TRUTHS
═══════════════════════════════════════════════════════════════════════════════

### 1. Typography Perceptual Truth
"Code Snippet typography feels like a focused editor window at 2AM — quiet chrome, semantic colors that let you read the code's grammar without reading every word, line numbers ghosted so far back they become a ruler you don't consciously see."

### 2. Spacing Perceptual Truth
"Code Snippet spacing feels like a well-curated museum label — generous enough to let the content breathe but restrained enough to feel purposeful, where every pixel of padding serves readability rather than decoration."

### 3. Container Perceptual Truth
"The Code Snippet container feels like a slate writing surface placed on a cream desk — grounded and technical where KortAI cards float like paper, but sharing the same philosophy of defining space through material, not decoration."

### 4. Hierarchy Perceptual Truth
"Code Snippet hierarchy feels like a museum exhibit where the artifact (code) is spotlit in the center, while the wall placard (language label) and exit sign (copy button) are visible but don't compete for attention."

### 5. Composition Perceptual Truth
"Code Snippet composition feels like a well-organized reference shelf — bottom-weighted with substance, topped by a quiet label, inviting extended study rather than quick glances."

### 6. Color Perceptual Truth
"Code Snippet color feels like a VS Code Dark theme that has been softened — familiar developer aesthetics with muted saturation, where syntax colors serve genuine comprehension but collectively create an IDE-like experience rather than editorial restraint."

### 7. Layout Perceptual Truth
"Code Snippet layout feels like a magazine pull-quote for code — confident editorial placement with just enough structure to say 'this is special' without becoming a bordered prison."

## NAMED CHARACTER
═══════════════════════════════════════════════════════════════════════════════

**Character Name:** The Precise Transcriptionist

**Archetype:** Faithful reproduction without interpretation

**Why This Name:**
- Transcriptionist: presents verbatim, doesn't editorialize
- Precise: character-level alignment, exact form
- Silent utility: copy button is the only gesture

**How to Stay in Character:**
When designing this component, ask: "Would a meticulous transcriptionist add this flourish?" If the answer involves decoration, personality, or explanation — remove it. The code speaks; the Transcriptionist merely ensures it's legible.

**Design Mantras:**
- "Accuracy over expression"
- "The code is the message"
- "One action, one outcome"

**Uniqueness Verified:** Does not conflict with:
- The Unhurried Editor (KortAI overall)
- The Watchful Guardian (warnings)
- The Helpful Neighbor (tips)
- The Wise Elder (essence)

## KORTAI ALIGNMENT ANALYSIS
═══════════════════════════════════════════════════════════════════════════════

### Alignment Level: PARTIAL (77%)

### Aligned Elements
- Typography restraint (3 treatments, single weight)
- Spacing generosity (16px padding, 1.6 line-height)
- Sharp corners (0px radius)
- Calm, unhurried energy
- Hierarchy through position
- Ghost-style copy button

### Documented Deviations

| Deviation | Reason | Severity |
|-----------|--------|----------|
| Dark background (#1E1E1E) | Code blocks conventionally dark; creates focus | MEDIUM |
| 6 syntax colors | Functional for comprehension; industry convention | HIGH |
| Missing red accent | No obvious use for single red accent in code | LOW |
| Color-primary hierarchy | Syntax highlighting requires chromatic variation | MEDIUM |

### Path to VERY CLOSE Alignment (90%+)

If pursuing full KortAI alignment:
1. Switch to warm cream background (#FEF9F5)
2. Reduce syntax to grayscale + red only
3. Keywords: #E83025 (red)
4. Comments: #888888 (gray, italic)
5. Everything else: graduated grays

**Decision:** Current implementation accepted as "appropriate domain deviation" — code display has different requirements than editorial content.

## ANTI-PATTERNS TO AVOID
═══════════════════════════════════════════════════════════════════════════════

**Typography:** Variable-width fonts, bold syntax, decorative headers
**Spacing:** No padding, cramped lines, header merged with code
**Container:** Light backgrounds, rounded corners >12px, drop shadows
**Color:** Rainbow highlighting, neon colors, >5 syntax hues
**Hierarchy:** Buttons more prominent than code, competing headers

**Summary Statement:**
"Code Snippet soul is violated when the component draws attention to itself rather than presenting code with quiet authority."
