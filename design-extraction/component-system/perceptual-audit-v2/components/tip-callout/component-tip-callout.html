Tip Callout — Multi-paragraph
Tip

Keep your CLAUDE.md file under 500 lines. Anything longer and you risk overwhelming the context window with instructions instead of actual work.

Consider splitting large instruction sets into separate files and referencing them only when needed.

Tip Callout — In Context

When working with large codebases, context management becomes critical. The agent needs enough information to make informed decisions, but not so much that it becomes overwhelmed.

Tip

Start with a minimal context file and expand only when the agent makes mistakes due to missing information. This "expand on failure" approach keeps context lean.

The goal is to provide just enough context for the current task, not a comprehensive overview of everything the agent might ever need to know.
