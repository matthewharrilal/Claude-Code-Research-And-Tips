Challenge Callout — In Context

Understanding context constraints isn't just theoretical—it requires hands-on experimentation. The most effective practitioners have internalized these principles through practice, not reading.

Challenge

Test the external state principle

Run the same multi-step task twice: once relying on Claude's "memory" and once writing intermediate results to files. Compare the outcomes and document which approach produced more consistent results.

Most practitioners who complete this exercise find that external state wins decisively—but the experience of discovering this yourself is more valuable than being told.
