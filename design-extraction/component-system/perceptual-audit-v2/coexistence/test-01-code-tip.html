Coexistence Test 01 — Code Snippet + Tip Callout

When implementing the Ralph Wiggum Loop, you need to create a simple shell script that continuously feeds prompts to Claude. Here's the basic implementation:

bash
1# Ralph Wiggum Loop — continuous agent iteration
2while :; do
3  cat PROMPT.md | claude-code
4done
Tip

Add a sleep 2 between iterations to avoid rate limiting. You can also use --continue to preserve context across runs.

The loop will continue running until you manually interrupt it with Ctrl+C. Each iteration picks up where the previous one left off, thanks to the external state stored in your prompt file.
