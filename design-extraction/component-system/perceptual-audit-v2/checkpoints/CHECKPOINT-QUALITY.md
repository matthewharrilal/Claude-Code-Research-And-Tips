**STOP IMMEDIATELY if any of these occur:**

1. ⚠️ Using "similar to [previous component]" instead of fresh analysis

3. ⚠️ Not opening/viewing the component
4. ⚠️ Structural truths sounding identical across components

6. ⚠️ Not finding any new blindspots

## STRUCTURAL ANALYSIS DEPTH REQUIREMENTS
═══════════════════════════════════════════════════════════════════════════════

### Each Component MUST Answer:

**Zone Structure:**
1. What ZONES does this component have?
2. How are zones SEPARATED (whitespace? lines? color?)?
3. What is the zone proportion (% of component)?

**Density:**
4. How does this contribute to page DENSITY?
5. What is the density level (LOW/MED/HIGH)?
6. Where are the breathing points?

**Axis Behavior:**
7. How does this behave in HORIZONTAL layout?
8. How does this behave in VERTICAL layout?
9. What axis transitions exist?

**Structural Organization:**
10. How does this COMBINE with other components?
11. What ORGANIZATIONAL STRUCTURE does this create?
12. How does this enable FINDABILITY?
