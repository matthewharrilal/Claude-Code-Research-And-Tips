## KEY PRINCIPLES (Quick Reference)
═══════════════════════════════════════════════════════════════════════════════

### The 6 Universal Rules
1. **Sharp Edges** — `border-radius: 0` ALWAYS
2. **Flat Design** — `box-shadow: none` ALWAYS
3. **Limited Palette** — red + cream + black only
4. **Typography Hierarchy** — serif display, sans body, mono code
5. **Heavy Borders** — 2-4px minimum when bordered
6. **Whitespace as Design** — generous padding, 1.6+ line-height

### Master Character: The Unhurried Editor
- Editorial calm, nothing to prove
- Trusts reader intelligence
- Restraint over expression

### Structural Archetypes
- **Single-Purpose Block** — Code Snippet
- **Left-Flagged Aside** — All Callouts
- **Typography-as-Structure** — File Tree
- **Multi-Column Grid** — Decision Matrix
