Gotcha

This approach will fail silently if the API key is not set. Always validate environment variables before initializing the client, or you'll spend hours debugging a blank response.

Common Mistake

Using --amend after a failed pre-commit hook will modify your previous commit, not the failed one. The commit didn't happen, so there's nothing to amend.

Instead, fix the issue, re-stage your files, and create a new commit. This prevents accidentally destroying previous work.

Watch Out

The context window resets between sessions. External state files (PROGRESS.md, TODO.md) are essential for maintaining continuity across restarts. Without them, each session starts from scratch.
