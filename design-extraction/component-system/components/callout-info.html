Note

This feature requires Claude Code version 1.4 or later. Check your version with claude-code --version.

Reference

For the complete specification of CLAUDE.md syntax, see the official documentation. The format has evolved significantly since the initial release.

Background

The term "Ralph Wiggum Loop" originated from a Simpsons reference in the Claude Code community. The pattern was first documented by Matt Pocock in early 2025.

The name stuck because of the "I'm in danger" meme quality of running autonomous agents overnight.

Compatibility

MCP servers require Node.js 18 or later. The @modelcontextprotocol/server-* packages use ESM syntax and modern JavaScript features.
