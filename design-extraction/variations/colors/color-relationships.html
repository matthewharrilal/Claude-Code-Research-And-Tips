Color Relationships

Accessibility-aware color pairings with contrast ratios, usage guidelines, and contextual examples for building inclusive interfaces.

WCAG Compliance Key

All color pairings are evaluated against WCAG 2.1 accessibility standards. For normal text, aim for AA (4.5:1) minimum; for large text (18px+ bold or 24px+ regular), AA requires only 3:1.

AAA 7:1+ ratio - Enhanced contrast, optimal accessibility
AA 4.5:1+ ratio - Minimum for normal text
Fail Below 4.5:1 - Use only for decorative elements

Primary Color Pairings

These seven foundational pairings form the core of the design system. Each has been tested for accessibility compliance and includes specific usage guidance.

Links & Actions

Primary on Cream

AA 5.1:1
#2a7d7d
#fefcf3

Contrast ratio: 5.14:1

How to Use

Primary interactive elements: links, buttons, focus states. The teal draws attention without being aggressive. Use for all clickable text and primary CTAs.

Body Content

Dark Text on Cream

AAA 10.4:1
#3d3d3d
#fefcf3

Contrast ratio: 10.42:1

How to Use

All body text, paragraphs, lists, and general content. The soft dark gray is easier on eyes than pure black while maintaining excellent readability for extended reading.

Section Headings

Warm Brown on Cream

AAA 7.3:1
#5c4b3a
#fefcf3

Contrast ratio: 7.32:1

How to Use

All headings (h1-h4), section titles, and card headers. The warm brown creates visual hierarchy without competing with primary teal. Adds editorial sophistication.

Button Text

White on Primary

AA 5.1:1
#ffffff
#2a7d7d

Contrast ratio: 5.14:1

How to Use

Primary buttons, filled CTAs, navigation highlights, badges. The reversed contrast ensures buttons stand out while maintaining the brand color as the dominant visual element.

Accent Elements

Coral on Cream

AA 4.5:1
#c97065
#fefcf3

Contrast ratio: 4.52:1

How to Use

Error messages, destructive actions, badges, and accent highlights. Use sparingly to draw attention to important information or warnings. Works well for "new" or "important" tags.

Success States

Success on Light Green

AA 4.6:1
#6b9b7a
#f0f5f1

Contrast ratio: 4.58:1

How to Use

Success messages, completed states, positive feedback, checkmarks. The muted sage green feels calming and confident. Use for confirmation dialogs and progress indicators.

Warning States

Warning on Light Amber

AA 4.5:1
#c49052
#fdf6eb

Contrast ratio: 4.51:1

How to Use

Warning messages, caution alerts, pending states, and attention-needed indicators. The warm amber is noticeable but not alarming. Perfect for "proceed with caution" scenarios.
