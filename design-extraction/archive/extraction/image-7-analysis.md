## LAYOUT

### Grid Structure
- **Hero Area**: ~75% of page (giant typography)
- **Body Text**: Small columns below hero
- **CTA**: Button element
- **Banners**: Full-width strips at bottom

### Typography as Layout
- Text IS the design
- No images (except icons inline)
- Whitespace between lines

### Responsive Consideration
- Large type will dominate any viewport
- May need to stack differently on mobile

## KEY DESIGN PRINCIPLES DEMONSTRATED

1. **Outline/filled mixing**: Creates rhythm and hierarchy
2. **Character substitution**: Distinctive, memorable branding
3. **Mixed type families**: Condensed sans + elegant serif
4. **Typography as hero**: Text dominates, minimal imagery
5. **Inline icons**: Hand-drawn elements add personality
6. **Consistent palette**: Same colors across all brand materials
