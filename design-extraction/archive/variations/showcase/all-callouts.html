Design Variation Reference

1. Stamp / Seal

Official stamp aesthetic with rotated icons, double-ring seals, and textured backgrounds. Uses transform: rotate() for hand-stamped appearance. Best for: authoritative documentation.

2. Heavy Border

Brutalist/neo-brutalist with 8-10px left accent border, zero border-radius, and high-contrast colors. Uses Space Grotesk typography. Best for: developer-focused content.

3. Typography

Type hierarchy as design - 900-weight serif labels with light sans body. Minimal container styling lets typography speak. Best for: editorial/publication contexts.

4. Decorative Frame

Elegant corner ornaments via CSS pseudo-elements, refined serif labels, and sophisticated color palette. Best for: premium documentation experiences.
