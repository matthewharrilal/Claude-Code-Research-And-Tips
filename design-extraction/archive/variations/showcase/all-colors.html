Section 2 Color Relationships

Accessibility-aware color pairings with WCAG contrast ratios, usage guidelines, and contextual examples for building inclusive interfaces. All pairings meet at least AA compliance for normal text.

WCAG Compliance Key

All color pairings are evaluated against WCAG 2.1 accessibility standards. For normal text, aim for AA (4.5:1) minimum; for large text (18px+ bold or 24px+ regular), AA requires only 3:1.

AAA 7:1+ ratio - Enhanced contrast, optimal accessibility
AA 4.5:1+ ratio - Minimum for normal text
Fail Below 4.5:1 - Use only for decorative elements
