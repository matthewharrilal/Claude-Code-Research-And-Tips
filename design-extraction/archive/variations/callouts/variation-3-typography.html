Design Techniques

  • Label typography: Playfair Display at 24px, weight 900, uppercase with 0.08em letter-spacing
  • Content typography: Source Sans Pro at 16px, weight 400, line-height 1.7
  • Separator: 60px wide, 2px height, 40% opacity of semantic color
  • Container: 3px left border only - minimal chrome, maximum typography impact
  • Color strategy: Each semantic type uses a single coordinated color for label, separator, and border

Reference Sources

  • Brutalist: Bold uppercase headings, strong visual hierarchy, letter-spacing for impact
  • Editorial: Serif/sans pairing, refined separators, publication-quality typography mixing
  • NOUS Branding: Massive type creates instant hierarchy without decoration
  • Sanrok Headline: Typography as the primary design element, mixing weights intentionally
