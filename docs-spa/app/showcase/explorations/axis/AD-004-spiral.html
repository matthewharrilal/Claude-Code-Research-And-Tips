Use grid-template-columns: 1fr 0.618fr for the two-column variant, falling back to single-column below 768px viewport width.

Unresolved Questions

  • Does the angular spiral approximation lose essential properties of the mathematical Fibonacci spiral, or does the right-angle constraint actually strengthen the layout's cognitive clarity?
  • Can the border-weight gradient be extended beyond four levels (4px/3px/1px) without losing perceptual distinctness? At what point does fine-grained confidence differentiation become noise?
  • Would a reverse-spiral (starting at the outermost ring and spiraling inward) serve different documentation goals — for instance, an exploratory tutorial that narrows from broad context to specific technique?
  • How does the spiral layout interact with assistive technology? Do screen readers navigate the stratum sequence coherently when CSS presentation diverges from a conventional linear flow?
