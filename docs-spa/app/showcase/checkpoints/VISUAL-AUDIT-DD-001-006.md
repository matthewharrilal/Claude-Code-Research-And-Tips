### Issue #1: DD-001 Traffic-Light Violation

**Severity:** HIGH
**Location:** DD-001 dense-grid section
**Problem:** Green Tip callout immediately adjacent to Red Gotcha callout

**What I See in Screenshot:**
Looking at DD-001, in the "Token-Based Authentication" section, there is a 2-column layout with:
- Left column: Green-bordered Tip ("Store tokens in HTTP-only cookies...")
- Right column: Red-bordered Gotcha ("Token expiration must be checked...")

These are side-by-side with minimal spacing, creating a "Christmas lights" or "traffic light" effect that:
1. Reduces professional appearance
2. Creates visual vibration/tension
3. Violates anti-pattern registry

**Required Fix:**
- Option A: Add 48px+ vertical spacing between Tip and Gotcha
- Option B: Separate with neutral content between them
- Option C: Stack vertically instead of side-by-side

**Lines to Fix:** DD-001-breathing.html approximately lines 468-484
