### Sub-Agent A: Typography

**Mission:** Audit font hierarchy, typeface selection, weight distribution

**Questions:**
1. Is the correct font family used for this component type?
   - Serif (Instrument Serif) → Display text, Essence quotes
   - Sans (Inter) → Body text, general content
   - Mono (JetBrains Mono) → Code, File Tree
2. Does the size follow the type scale? (12px→48px in 9 steps)
3. Are font weights used appropriately? (400 regular, 500 medium, 600 semibold)
4. Is the special typography rule followed? (e.g., 10px uppercase labels)

**Soul Alignment Checklist:**
□ Uses correct font family for component type
□ Follows size hierarchy
□ Follows weight rules
□ Special rules followed (if applicable)

---

### Sub-Agent B: Spacing

**Mission:** Verify padding/margin consistency against spacing scale

**Questions:**
1. Does internal padding follow the 8px-based scale? (4, 8, 12, 16, 20, 24, 32, 40, 48, 64)
2. Is there consistent spacing between internal zones?
3. Does whitespace create appropriate rhythm?
4. Is semantic proximity correct? (related 8-16px, unrelated 24-48px)

**Soul Alignment Checklist:**
□ Uses spacing from scale
□ Internal spacing consistent
□ Creates appropriate rhythm
□ Semantic spacing correct

---

### Sub-Agent C: Containers

**Mission:** Audit boundaries, borders, backgrounds

**Questions:**
1. Is border-radius: 0 everywhere? (LOCKED - no exceptions)
2. Are borders the appropriate weight? (2-4px when present)
3. Is box-shadow: none? (LOCKED - no exceptions)
4. Does the container define clear zones?

**Soul Alignment Checklist:**
□ border-radius: 0 everywhere
□ Borders appropriate weight (2-4px if bordered)
□ Clear zone definition
□ No unauthorized shadows

### Sub-Agent D: Hierarchy

**Mission:** Examine visual weight distribution and information priority

**Soul Alignment Checklist:**
□ Clear reading order
□ Emphasis appropriate
□ No competing focal points
□ Hierarchy serves content

### Sub-Agent F: Color

**Mission:** Verify palette compliance and semantic color usage
