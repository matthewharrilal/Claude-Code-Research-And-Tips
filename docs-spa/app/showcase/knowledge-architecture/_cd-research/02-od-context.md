## 5. OD Anti-Patterns CD Must Avoid

### Anti-Pattern 1: 2px Border Epidemic (OD-F-AP-001)
- **What happened:** 108 CSS declarations using 2px borders across OD, generating 1,619 computed instances
- **Why it fails:** Violates COMP-F-010 (callout family DNA: 4px), creates visual inconsistency, reduces editorial authority
- **The fix:** 3-category border system (structural 3px+, separator 2px, micro 1px)
- **CD action:** Enforce 3-category system from day one. Audit every `border:` declaration for width. Zero 2px structural borders.

### Anti-Pattern 2: Semi-Transparent Backgrounds (Systemic)
- **What happened:** rgba < 1.0 and opacity < 1.0 found in EVERY re-enrichment wave (~32 violations across all 6 ODs)
- **Binary rule:** opacity === 1.0 on all visual elements. No exceptions.
- **CD action:** Audit `rgba(` and `opacity:` explicitly. This is a binary gate, not judgment call.

### Anti-Pattern 3: ANTI-PHYSICAL Identity Violations (20/20 SOUL FAILs)
- **Pattern:** The system rejects anything making flat screens simulate 3D physical space
- **Categories rejected:** (a) decorative depth/shadows, (b) animated z-movement, (c) rounded shapes suggesting physical objects, (d) gradients suggesting light/material, (e) traffic-light color semantics, (f) parallax implying z-space
- **CD action:** Use as pre-filter for ALL proposed combination techniques. If a technique creates perceived depth/movement/3D, it will FAIL soul compliance.

### Anti-Pattern 4: Deferred Provenance
- **What happened:** OD deferred writing finding IDs, causing the OD-F-005 collision requiring 18-agent remediation
- **CD action:** Write CD-F-### entries DURING build, not after. If you discover something, document it immediately.

### Anti-Pattern 5: Building Before Conventions
- **What happened:** OD built explorations in sequence without unified conventions, creating 3 quality dialects requiring 43-agent remediation
- **CD action:** Establish CD-CONVENTION-SPEC BEFORE CD-001. Apply OD-CONVENTION-SPEC.md as baseline plus CD-specific additions.

### Anti-Pattern 6: No Crown Jewel Plan
- **What happened:** OD-006 emerged organically as synthesis page, which worked but was unplanned
- **CD action:** Designate CD-006 as crown jewel from day one. Plan research budget and compound combination techniques accordingly.

### Anti-Pattern 7: Dialect Drift
- **What happened:** Sequential building created Polished (OD-001/002), Functional (OD-003/004/005), and Editorial (OD-006) dialects
- **CD action:** All 6 CDs share baseline convention. Build order should NOT create quality variation.

### Anti-Pattern 8: Mode-Transition Without Breathing (OD-F-007)
- **What happened:** OD-006 revealed that switching between organizational modes requires 48px+ breathing zones and a visual signal (divider, header, or mode indicator)
- **CD action:** CD will inherently combine modes. EVERY mode transition needs a breathing zone to prevent "cognitive whiplash." This is arguably CD's MOST CRITICAL inherited anti-pattern to respect, since CD's entire purpose is combining patterns.
