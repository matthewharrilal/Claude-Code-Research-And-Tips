## BUILDER-1B DISCOVERIES (Appended during re-enrichment execution)

### DISCOVERY-B1B-001: Progressive Disclosure Labels as Narrative Wayfinding

**Finding:** Adding explicit "Disclosure Level" labels before each code block (e.g., "Disclosure Level: Naive Implementation", "Disclosure Level: Schema Approach", "Disclosure Level: Production Architecture") creates a secondary wayfinding system parallel to the tension meter. The labels make the hermeneutic code (EXT-NARR-003) VISIBLE to the reader — they know the information is being strategically revealed, which transforms strategic withholding from an invisible technique into a transparent pedagogical contract. This is more effective than invisible withholding because the reader becomes an active participant in the progressive disclosure rather than a passive recipient.

**Relationship to existing findings:** Extends EXT-NARR-003 (information withholding) and EXT-NARR-009 (progressive disclosure = narrative arc). Where those findings describe the TECHNIQUE, this discovery addresses TRANSPARENCY — making the technique visible to the reader.

**Identity tension:** PRODUCTIVE. The Unhurried Editor is authoritative and curated (identity statement). Making the disclosure structure visible is an act of editorial confidence — "I am revealing this in the optimal sequence, and I want you to know that." This resolves EXT-NARR-003's productive tension between editorial authority and strategic withholding: the editor is not being cagey, they are being pedagogically transparent.

### DISCOVERY-B1B-002: Freytag Falling Action Solves the "Summary Problem"

**Finding:** The original OD-002 (3 acts + resolution) placed the file tree visualization and the lesson/challenge in Act IV "Resolution," which combined falling action and resolution into one act. Separating them per Freytag (EXT-NARR-001) into Act IV (Falling Action = architecture view + lesson) and Act V (Resolution = reflection + summary) produces measurably better cognitive decompression. The falling action (Act IV) lets the reader process what they learned through a structural lens (file tree, challenge), while the resolution (Act V) provides the meta-reflection ("the journey is the valuable thing to document"). Without this separation, readers exited the dense climax directly into a summary, which felt like an abrupt tonal shift. The Freytag model's falling action serves as a cognitive decompression chamber.

**Relationship to existing findings:** Directly validates EXT-NARR-001 (Freytag 5-beat). Extends OD-F-009 (narrative creates CRESCENDO) — the CRESCENDO now has a proper exhale, not just a cut-off.

### DISCOVERY-B1B-003: Convention Spec Header Transform Changes Page Identity

**Finding:** Converting from a white header with colored bottom border to a full-bleed dark (#1A1A1A) header per Convention Spec Section 4 dramatically changed OD-002's page identity. The dark header creates a stronger editorial entrance that anchors the CRESCENDO — the page starts in a visually confident place, which makes the sparse exposition (Act I) feel like an intentional contrast rather than a blank beginning. The EXT-NARR-008 "four pillars" analysis was correct: the Layout pillar gap was the single biggest quality gap. The dark header + inner wrapper pattern immediately addresses it.

**Relationship to existing findings:** Validates EXT-NARR-008 SURPRISE-3 (OD-002 missing two of four editorial pillars). The Layout pillar gap is now addressed.

### DISCOVERY-B1B-004: Type Scale Convention Alignment Strengthens CRESCENDO

**Finding:** Switching from hardcoded px values (28/24/22/20) to the Convention Spec's 5-level type scale (--type-page 40px / --type-section 26px / --type-subsection 22px / --type-body 16px) created a MORE perceptible CRESCENDO typography cascade. The original 28→20 range was only 8px total reduction across 4 acts. The convention spec scale creates a 40→16 range (24px reduction across 5 acts), which is dramatically more visible. This validates EXT-NARR-005's recommendation for "minimum 4px differential per act" — the convention spec scale inherently provides this.

**Relationship to existing findings:** Validates EXT-NARR-005 (typographic hierarchy as density signal) and OD-F-010 (typography size reduction mirrors density). The convention spec type scale is inherently CRESCENDO-aligned for narrative content.
