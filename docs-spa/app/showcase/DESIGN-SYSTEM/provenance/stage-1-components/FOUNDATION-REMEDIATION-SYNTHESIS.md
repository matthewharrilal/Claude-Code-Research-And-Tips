### Why 1 Iteration Per Component (Not 5)?

The perceptual deepening protocol specifies **minimum 5 iterations** when components
are failing and need iterative refinement. In this audit, all components were found
to be **already passing** on first visual inspection.

The protocol allows **early exit when:**
1. Squint test shows same visual weight as reference
2. All soul tokens are verified
3. Character is correctly expressed

Since components had been remediated before this session, no additional iterations
were needed to achieve convergence.
