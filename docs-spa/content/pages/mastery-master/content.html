Orchestration Patterns

Ralph Pattern: Autonomous Development Loops
Fresh context, external state, overnight execution
The persistence pattern for AI coding agents. At its core: while :; do cat PROMPT.md | claude ; done. Ralph spawns fresh Claude instances repeatedly, using external files (prd.json, progress.txt) for memory. Each iteration reads state, picks one task, implements it, commits, and exits. Enables "ship while you sleep" autonomous development.
Level 3-4 5 min setup $10-50/run
