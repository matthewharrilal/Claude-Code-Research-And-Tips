Intelligence System: Beads

Agents suffer from "50 First Dates" syndrome -- they wake up with no memory of yesterday. Beads provides git-backed persistent intelligence, universal context sharing, hash-based IDs to prevent conflicts, and memory decay through compaction of closed operations.

Campaign Timeline: 6 Waves

Strategic Evolution
Waves 1-3 -- Initial Campaigns

Manual to Chat-Based Operations

Traditional coding (1x baseline) through completion-based support (2x) to chat-based CHOP (5x). The lone-operator era. Each wave improved individual capability but never questioned the single-operative model.

Waves 4-5 -- Force Multiplication

Agents to Agent Clusters

Coding agents (10x) and multi-agent parallel formations (25x). The squad begins to form. Individual operatives are capable; coordinated squads are decisive.

Wave 6 -- Full Theater Operations

Agent Fleets: AI Officers Managing Formations

Gas Town is Wave 6 infrastructure. AI officers managing agent formations. 50x+ operational capacity. The command post doesn't just deploy agents -- it IS agents commanding agents, coordinated by AI officers, with a human CO at the apex.

Standing Orders

Order No. 1 -- GUPP

Universal Propulsion

"If there is work on your hook, YOU MUST RUN IT." Work persists in git worktrees. Agents check hooks on startup. If work exists, execution is immediate. No confirmation. No waiting.

Order No. 2 -- Zero Framework

Cognitive Delegation

"Claude makes all decisions. The orchestrator executes." All tactical decisions delegated to AI. No heuristics, regex, or parsing in the infrastructure. The framework provides logistics, not strategy.

Order No. 3 -- Cattle, Not Pets

Personnel Doctrine

Individual sessions are expendable. The mission plan is persistent. The operative is not the deployment -- the operative is a Bead with a global identity. Rotate freely.

Order No. 4 -- Nondeterministic

Operational Resilience

Path is fully nondeterministic. Different deployments take different routes. Operations complete as long as you keep deploying operatives. Resilient to individual unit failures.
