Civic Center District

Zone: Town Level

The town's governing body. Three civic institutions operate at the global level -- one of each, singular authority. All municipal decisions route through this district.

City Hall -- Central Authority

Mayor

Chief coordinator and your concierge. Receives high-level directives from you (the resident Overseer), routes work to appropriate rigs, manages cross-district coordination, handles town-level decisions. Everything in the town flows through City Hall.

Industrial District

Underground Utilities: Beads

Subsurface Data Infrastructure -- 225K Lines Go

Every building in Gas Town is connected by underground pipes. Agents suffer from "50 First Dates" syndrome -- they wake up with no memory of yesterday. Beads provides the persistent plumbing: git-backed storage, universal context, hash-based IDs, and memory decay through compaction.

Main Line -- Hot Context

  • Current task state and active operations
  • Last N interactions across workers
  • Active file references

Secondary Pipe -- Warm Context

  • Recent commit history
  • Cross-agent message logs
  • Rig state snapshots

Archive Pipe -- Cold Context

  • Full project history
  • Previous run outcomes
  • Archived decisions

Progress Parkway: 6 Waves

Development Timeline
Waves 1-3 -- Small Town Era

Manual to Chat-Based

Traditional (1x) to completions (2x) to chat-based CHOP (5x). The cottage-industry era. Individual craftspeople, no municipal infrastructure.

Waves 4-5 -- Township Formation

Agents to Agent Clusters

Coding agents (10x) and multi-agent clusters (25x). The village grows into a town. Individual workshops give way to organized districts.

Wave 6 -- Full Municipality

Agent Fleets: The Incorporated Town

Gas Town is Wave 6 infrastructure. A fully incorporated municipality with zoned districts, appointed officials, public services, and a resident human mayor. 50x+ output.

Town Ordinances

Operating Charter
Ordinance 1 -- GUPP

Universal Propulsion

"If there is work on your hook, YOU MUST RUN IT." Work persists in git worktrees. Agents check hooks on startup. Execution is immediate.

Ordinance 2 -- Zero Framework

Cognitive Delegation

"Claude makes all decisions." All intelligence delegated to AI. The municipal framework provides infrastructure, not governance logic.

Ordinance 3 -- Cattle, Not Pets

Residency Policy

Individual sessions are transient residents. The work graph is permanent infrastructure. The agent is a Bead with a town address.

Ordinance 4 -- Nondeterministic

Resilience Code

Routes vary. Different runs take different streets. The town functions as long as workers keep showing up. Resilient to individual failures.

Zoning Restriction -- Entry Requirements

"If you're not at Stage 7, you'll get your face ripped off by superintelligent chimpanzees." Residency in Gas Town requires Stage 7+ classification, tmux proficiency, $50-200/day municipal tax, high chaos tolerance, and 100+ hours of multi-agent experience. No exceptions.
