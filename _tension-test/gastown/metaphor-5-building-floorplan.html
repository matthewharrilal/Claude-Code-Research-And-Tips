Basement -- Mechanical / Data Infrastructure Level

Beads: Building Infrastructure

Mechanical + Data Systems -- 225K Lines Go

Every room in the building is connected through basement infrastructure. Agents suffer from "50 First Dates" syndrome -- they arrive at their desk with no memory of yesterday's work. Beads provides the persistent systems: git-backed storage as the data backbone, universal context through shared conduits, hash-addressed locations to prevent collision, and garbage collection through compaction.

Construction Phases Building Evolution
Phases 1-3 -- Shed to Cottage

Single-Room Structures

Manual coding (1x tool shed) to completions (2x workshop) to chat-based (5x cottage). Each phase enlarged the room, but it was always one room. No hallways, no plumbing.

Phases 4-5 -- Cottage to Office Block

Multi-Room Construction

Coding agents (10x small office) to multi-agent clusters (25x office suite). Multiple rooms share a hallway. Coordination emerges. Plumbing gets complicated.

Phase 6 -- Full Facility

Purpose-Built Multi-Story Building

Gas Town is Phase 6 construction. A purpose-designed facility with specialized floors, dedicated infrastructure, security, and a human building manager. 50x+ capacity. The building manages itself -- you just file the work orders.

Building Codes Operating Regulations
Code 1 -- GUPP

Universal Propulsion

"If there is work on your hook, YOU MUST RUN IT." Work orders persist in mailboxes (git worktrees). Staff check their inbox on arrival. If work exists, start immediately.

Code 2 -- Zero Framework

Cognitive Delegation

"Claude makes all decisions." All intelligence belongs to the occupants, not the building. The facility provides rooms and corridors, not management logic.

Code 3 -- Cattle, Not Pets

Occupancy Policy

Individual staff are temporary occupants. The building is permanent. The worker is a name on a door, not a structural element. Reassign offices freely.

Code 4 -- Nondeterministic

Flexible Routing

Staff take different hallways each day. Different shifts use different routes. The building functions as long as rooms are occupied. Tolerant of individual absences.
