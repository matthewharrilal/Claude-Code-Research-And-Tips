# Grammar Layer

**Files:**
- `mechanisms.md` — 20 compositional mechanisms organized by category (spatial, hierarchy, component, depth, structure) + transition grammar + restraint guidance
- `grammar.md` — Compositional rules: nesting, rhythm, zones, density, component sequencing, organizational patterns, density patterns, responsive, anti-patterns

**Phase-gated:** Available at Phase 4+ (not before metaphor lock).

**Key concepts:**
- Mechanisms are CSS TECHNIQUES that transfer across metaphors (pass Name Test + Transfer Test)
- Grammar rules are compositional CONSTRAINTS (max nesting depth, density ceilings, sequencing)
- Transition types: SMOOTH / BRIDGE / BREATHING
