# Design Variation Comparison Document

A systematic comparison of all visual variations to verify distinctiveness and prevent confusion.

---

## Code Block Variations

| Aspect | Terminal | Zine | Brutalist | Editorial |
|--------|----------|------|-----------|-----------|
| **Background** | Black (#0d1117) with radial glow | Cream newsprint (#f5f5dc) | Pure black (#000) or white | Warm cream (#faf8f5) |
| **Border** | CRT bezel with gradient shadows | Dashed border, corner marks | 10px solid left border only | 1px thin refined border |
| **Texture** | Scanlines + vignette overlay | Xerox grain (SVG noise) | None - clean surfaces | None - smooth |
| **Typography** | VT323 monospace + phosphor glow | Courier New typewriter | System monospace | JetBrains Mono + Playfair Display serif |
| **Rotation** | 0deg (squared CRT frame) | -1.5deg (hand-placed feel) | 0deg (rigid, intentional) | 0deg (editorial precision) |
| **Key Features** | Traffic light buttons, blinking cursor, green phosphor glow | Rubber stamp header, page numbers, corner tape marks | Massive uppercase title above block, no shadows | Italic serif caption, hover-reveal copy button, line numbers |
| **Distinctive?** | YES | YES | YES | YES |

### Could any two be confused?

**NO** - Each variation uses fundamentally different visual approaches:

- **Terminal vs Zine**: Terminal is dark/glowing/digital; Zine is cream/textured/analog
- **Terminal vs Brutalist**: Terminal has heavy decoration (scanlines, glow, bezel); Brutalist strips all decoration
- **Terminal vs Editorial**: Terminal uses monochrome green; Editorial uses warm cream with serif typography
- **Zine vs Brutalist**: Zine uses rotation/texture/dashed borders; Brutalist is rigid/clean/sharp
- **Zine vs Editorial**: Zine is rough/DIY; Editorial is refined/elegant
- **Brutalist vs Editorial**: Brutalist uses black/white stark contrast with massive type; Editorial uses warm cream with typography mixing

## Recommendations for Additional Differentiation

While all variations are currently distinct, these enhancements could strengthen uniqueness if needed:

### Code Blocks
- Terminal: Consider adding subtle animation to scanlines
- Zine: Could increase rotation to -2 or -2.5deg
- Editorial: Currently excellent - no changes needed

### Cards
- Typography: The extreme version (background level number) is most distinctive
- Badges: Very strong - the most unique of all card variations
- Textured: Could add subtle color tinting to the grain

### Callouts
- Stamp: The most distinctive due to rotation - no changes needed
- Heavy Border: Consider increasing contrast between border and background
- Typography: Could use even larger label (28-32px) for more impact
- Decorative: Corner ornaments could be more elaborate if needed
