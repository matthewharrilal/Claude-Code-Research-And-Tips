# Image 6 Analysis: Sanrok Red Team Page

## Overview
Editorial/magazine aesthetic with bold red and cream palette, mixed typography (condensed sans + elegant italic serif), duotone photography, and playful elements like emoji and arrow motifs. Premium but approachable.

---

## COLORS

### Primary Palette
| Name | Hex | Usage |
|------|-----|-------|
| Sanrok Red | `#E63946` | Primary, banners, photo overlay |
| Warm Cream | `#FEF5E7` | Background, cards |
| Off-White | `#FDFBF7` | Lighter background areas |
| Pure White | `#FFFFFF` | Text on red, card backgrounds |
| Charcoal Black | `#1A1A1A` | Text on light backgrounds |

### Color Usage
- Red dominates (~40% of page)
- Cream provides warmth
- White for clean card elements
- Black for body text
- **No other colors** - strict palette discipline

## IMAGERY

### Team Photos
- **Treatment**: Duotone (Red + White/Cream)
- **Style**: Candid/personality shots
- **Overlay**: Red tint removes color, creates cohesion
- **Crop**: Various (some square, some with overlay elements)

### Photo Overlays
- Message bubble elements ("My mom said our studio is the best.")
- Positioned over photos
- White background, small text
- Adds personality and humor

### Duotone Technique
- Original colors removed
- Red replaces shadows
- Cream/white replaces highlights
- Creates unified visual language

---

## DECORATIVE ELEMENTS

### Arrow Motifs (→ ←)
- Used in URL banner
- Flank text symmetrically
- Simple directional emphasis
- Adds dynamic quality

### Emoji/Symbol Usage (ツ)
- Japanese katakana as emoji substitute
- Playful, unexpected
- Cultural reference
- Softens professional tone

### Star Symbol (☆)
- Logo/brand indicator
- Before "SANROK" in nav
- Simple brand mark

### Message Bubbles
- Chat/message app aesthetic
- Overlaid on images
- Add humor and humanity
- "sent ✓" delivery indicators

### Numbered Team Cards
- "1", "2", "3" indicators
- Small, subtle
- Organizational element

### Banner System

## KEY DESIGN PRINCIPLES DEMONSTRATED

1. **Mixed typography**: Condensed sans + elegant serif in same line
2. **Duotone photography**: Unified visual language
3. **Playful elements**: Emoji, chat bubbles, personality
4. **Full-width banners**: Rhythm and emphasis
5. **Limited palette discipline**: Red + cream + white + black only
6. **Arrow motifs**: Dynamic, directional energy
