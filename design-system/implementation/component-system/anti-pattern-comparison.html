The Soul Test

Questions to ask before implementing any component

  1. "Does this look like it could appear in a magazine?" ‚Äî If yes, pass.
  2. "Does this look like Bootstrap?" ‚Äî If yes, fail.
  3. "Could I swap the accent color to Bootstrap blue and it would fit?" ‚Äî If yes, fail.
  4. "Does every design choice serve hierarchy or readability?" ‚Äî If yes, pass.
  5. "Is there any decoration that doesn't earn its place?" ‚Äî If yes, fail.

If a component could exist in Bootstrap, Tailwind, or Material without modification, the implementation has failed. Our components must be unmistakably editorial.
