## Recommendation Tiers

### Tier 1: Exact Match (Reference Images)
**Right Didone** - Most likely the font in the reference screenshots, or its closest commercial equivalent. Modern Didone with grotesque-influenced proportions, multiple width sub-families.

### Tier 2: Actual Implementation (Sanrok Live Site)
**Instrument Serif** - Confirmed as the font used on Sanrok's actual website. Single weight limits flexibility but guarantees authenticity to the live site.

### Tier 3: Premium Alternatives
- **Noe Display** - Different character (wedge serifs) but excellent quality
- **Domaine Display** - More transitional, less sharp, but refined

### Tier 4: Free Alternatives
- **Bodoni Moda** - Best free option; more classical than reference
- **Libre Bodoni** - Simpler free Bodoni; traditional feel
- **Abril Fatface** - Too bold/theatrical for this aesthetic

---

## Decision Guide

### If Budget Allows (~$200-500):
**Primary:** Right Didone (one sub-family + essentials)
**Backup:** Instrument Serif (free) for secondary headlines

### If Budget Limited (~$60-150):
**Primary:** Domaine Display (select weights)
**Backup:** Instrument Serif (free)

### If Zero Budget:
**Primary:** Instrument Serif (matches live site exactly)
**Support:** Bodoni Moda (for situations needing weight variety)

### If Maximum Flexibility Required:
**Primary:** Right Didone (full collection with variable fonts)
- 98+ styles covering all width/weight combinations
- Variable font axes for fine-tuning
- Cyrillic support if needed
