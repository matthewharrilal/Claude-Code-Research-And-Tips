Elevation 1-2 -- Foothills

Elevation 3 -- The Treeline

Autonomy: The Ralph Loop

Above the treeline, the terrain changes. The Ralph Loop grants autonomous climbing. Fresh context at every camp. State preserved in cairns between stages. You can now set a course and let the expedition continue while you rest.

#!/bin/bash
MAX_ITERATIONS=${1:-25}

for i in $(seq 1 $MAX_ITERATIONS); do
  echo "=== Camp $i ==="
  OUTPUT=$(cat prompt.md | claude --dangerously-skip-permissions)
  if echo "$OUTPUT" | grep -q "<promise>COMPLETE</promise>"; then
    echo "Summit reached!" && exit 0
  fi
done
