## 4. CONTENT-FORM FIT ASSESSMENT

### Using Gap 7 Criteria (semantic-rules.md)

**Q: Does the content have enough substance for flagship-tier form decisions?**

YES. The content has:
- 10 distinct tables (each requiring unique visual treatment decisions)
- 5 unified principles (each with synthesis paragraph + evidence + implementation guidance)
- 3 conflict resolutions (each with thesis/antithesis/synthesis structure)
- 4 component sequences (each with flow diagrams)
- 15 exploration hypotheses (requiring compact presentation)
- Cross-references between research streams (R1+R3, R2+R4, etc.)

Every form decision has a content-driven justification:
- Table styling varies because tables encode DIFFERENT semantic types (dimensions vs patterns vs components vs hypotheses)
- Spacing varies because sections serve DIFFERENT reader modes (learning principles vs resolving conflicts vs looking up findings)
- Typography varies because content roles differ (synthesis wisdom = serif, implementation guidance = sans, code signatures = mono)

**Q: Are there natural tension points between form and meaning?**

YES, at least 4:

1. **Network vs Sequence:** The 5 principles are INTERCONNECTED (R-1 + R-3 = Principle 1, R-2 + R-4 = Principle 2, etc.) but HTML is SEQUENTIAL. The form must communicate interconnection through a sequential medium. Tension-resolution: cross-reference annotations, shared color encoding between related research streams, visual connectors.

2. **Celebration vs Restraint:** The Master Insight is a moment of intellectual triumph ("337 findings compressed to 5 principles!"). The soul demands restraint. Tension-resolution: Essence Callout with serif italic -- warmth within sharp edges.

3. **Dense Reference vs Flowing Narrative:** The principles alternate between dense tables (reference mode) and flowing synthesis (narrative mode). The form must handle BOTH without jarring transitions. Tension-resolution: zone backgrounds shift between table-mode and narrative-mode sections, with breathing zones at transitions.

4. **Individual Streams vs Unified Vision:** Each research stream (R1-R5) has its own identity, but the synthesis page must present a UNIFIED vision. The form must honor individual contributions while serving the unified framework. Tension-resolution: research stream color coding that fades into unified principle color.

**Q: Is there risk of form overwhelming content?**

MODERATE -- addressable. The primary risk vectors:

| Risk | Severity | Mitigation |
|------|----------|-----------|
| Too many zones for content volume | MEDIUM | Gap 6 Rule 3 verified: ~2,700 words supports 3 zones (max 4). Do NOT exceed 4. |
| Metaphor over-labeling | HIGH | Ceiling experiment's dominant failure. Builder Warning W-IMPLICIT applies. Metaphor must be STRUCTURAL (backgrounds, borders, spacing) not TEXTUAL (zone labels, metaphor vocabulary). |
| Table monotony | MEDIUM | 10 tables risk visual sameness. Vary table treatments: some full-width, some in bento cells, some with border-weight encoding. |
| Self-referential excess | LOW | Content is about design principles. The temptation to demonstrate EVERY principle on the page is strong. Cap demonstrations at 3-4 principles visually demonstrated; explain others textually. |

### Highest-Priority Risk: Table Monotony

With 10 tables on a single page, the strongest risk is that the page becomes a table gallery. The mitigation must be ARCHITECTURAL, not cosmetic:

1. **Categorize tables by function:**
   - Reference tables (Top 10, Anti-Patterns): full-width, scannable, F-pattern
   - Dimensional tables (6 dimensions, 5 patterns): compact, bento-style
   - Character tables (11-row components): featured, DENSEST moment, visual climax
   - Hypothesis tables (3x5): compact grid, atmospheric, de-emphasized

2. **Vary table treatments:**
   - Border-weight encodes importance (4px for character table, 1px for hypotheses)
   - Background differentiates function (warm cream for principles, subtle shift for reference)
   - Width varies (some full-container, some 75% with narrative alongside)

3. **Intersperse with non-tabular content:**
   - Synthesis narratives between principle tables
   - Resolution narratives between conflict resolutions
   - Code block density signatures break tabular rhythm
   - Pullquote (Master Insight) provides serif-italic relief
